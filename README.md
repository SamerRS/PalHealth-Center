PalHealth Center is a web-based platform built using Django to help patients and doctors manage appointments efficiently. It is designed to streamline the process of booking, tracking, and managing healthcare appointments while providing an intuitive and secure interface for both patients and administrators.

Features
Patient Management:

Patients can register, log in, and manage their profiles.
Patients can search for doctors and book appointments.
Doctor Management:

Admins can manage doctor profiles, including specializations and availability.
Appointment Management:

Patients can book and manage appointments with doctors.
Admins can view and manage all appointments, including updating their statuses.
Responsive Design:

The platform is mobile-friendly, ensuring a seamless user experience across all devices.
Secure Authentication:

Passwords are securely hashed, and forms have validation to protect user data.
Dynamic Interaction:

Uses AJAX to provide real-time feedback and dynamic updates for appointment availability.
Tech Stack
Backend: Django (Python)
Frontend: HTML, CSS, Bootstrap
Database: MySQL
Hosting: AWS (Amazon Web Services)
Version Control: Git & GitHub
